## How to run

Open the terminal and run the following command:

```bash
./run.sh <file_name>
```

For example, to run the `TwoSum.cpp` file under the `0.ArraysAndHashing` directory, run the following command:

```bash
./run.sh 0.ArraysAndHashing/TwoSum.cpp
```
